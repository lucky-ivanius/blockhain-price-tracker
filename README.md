# Blockchain price tracker

## Installation

```bash
$ pnpm install
```

## Environment variables

```bash
$ cp .env.example .env
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Run with Docker

```bash
$ docker-compose up -d
```
